## About

AAA Client is a lightweight Minecraft 1.8.9 Forge client focused on:

* Performance
* Responsiveness
* PvP utilities
* HUD customization
* Quality-of-life features

Unlike clients such as Lunar, Badlion, Feather, or Soar, AAA Client is designed as a Forge mod rather than a custom launcher.

The goal is simple:

> Improve the game experience without replacing Minecraft itself.

No account system.
No launcher.
No unnecessary background processes.
Just a Forge client.

## Philosophy

AAA Client follows a few simple rules:

### Performance First

Every feature should justify its resource usage.

### Keep It Lightweight

No launcher.
No bloat.
No random background services.

### Respect Server Rules

AAA Client is **not** intended to provide unfair advantages.

The project does not include:

* Cheats
* Packet exploits
* Lag switches
* Velocity manipulation
* Reach modifications
* Combat automation
* Server bypasses

If a feature would get you banned, it doesn't belong here.

### Open Development

The code is intended to be readable, maintainable, and easy to extend.
